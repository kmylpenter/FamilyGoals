# FamilyGoals

Webapp do śledzenia celów finansowych rodziny.

## Funkcje

- PWA (Progressive Web App) - działa offline
- Zabezpieczenie PIN-em
- Śledzenie celów oszczędnościowych
- Dostęp z każdego urządzenia

## Technologie

- Frontend: HTML/CSS/JS
- Storage: LocalStorage (offline-first)
- PWA: Service Worker

## Status

Aktywny rozwój. Szczegóły w `logs/STATE.md`.
